# template-js
Template for modern ECMAScript projects

# how to use
I recommend using the [bun](https://bun.sh/) package manager.
```bun i && bun dev```
Otherwise:
```npm i && npm run dev```

# why
- Simple
- Lightweight
- Production ready
- Following SPA standards
- Following best practices for rapid web development

Built with:
- Vite
- Express.js
- React
- TailwindCSS
